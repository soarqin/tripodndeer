# M0.2 · 边索引地图格式（Edge-Indexed Map Format）

## TL;DR

> **Quick Summary**：用边索引格式（DCEL/Half-Edge 思路）取代 per-cell polygon，实现物理上不可能的零缝隙。每条共享边只存一次，cells 通过 ID 引用。Cubic-Bezier 曲线（SVG 标准）支持未来编辑器与图像导入工作流。
>
> **Deliverables**：
> - 新数据格式（edges 全局表 + sites 通过 boundary 引用）
> - cubic-bezier 渲染（带 Catmull-Rom-to-Bezier 控制点计算）
> - 邻接运行时推导（不再存储）
> - 重写 generator + factory + tile-cache + tests
> - 重新生成 sites.json + 重录 demo
>
> **Estimated Effort**：Medium（5 任务，4-6 小时）
> **Critical Path**：T1（schema）→ T2（generator）→ T3（factory）→ T4（renderer）→ T5（E2E + 重录）→ F1-F4

## Context

### Original Request
M0.1 演示后用户反馈：
> 地块之间的缝隙还是比较严重。需要设计一个合理的地图配置格式，便于后续使用编辑器画地块分割线或者直接从地图册图片生成。建议采用线段（即包含多个锚点和曲率的曲线）"包围"的模式来记录各个区块，同一线段的索引可以被多个地图所使用。

### Diagnosis（M0.1 缝隙根因）

`tools/generate-m0-map.ts` 的 `perturbEdge` 函数**方向不对称**：
- Hash 是 order-independent ✅
- PRNG 序列两侧 cell 相同 ✅
- 但 `t = j/N` 在不同方向遍历时，**空间位置不同** ❌
- 法线 `perp(a,b) = -perp(b,a)` ❌

→ 两侧 cell 在不同空间位置应用相同噪声 → 真实几何缝隙。

### 决策：根本性重构（不是补丁修 perturbEdge bug）

**方案选择：**

| 方案 | 评估 |
|-----|-----|
| 修 perturbEdge bug（M0.1.1 补丁） | 治标 — cells 仍持有 polygon 副本，浮点精度仍可能漂移；编辑器/图像工作流仍需重写 |
| **边索引格式（M0.2，本计划）** | 治本 — 单一真相来源；编辑器/图像 workflow 自然兼容；GIS/SVG 标准做法 |
| 单画布合成（M2+ 方案） | 过度工程 — M0 不需要 |

### 设计文档参考
- `docs/design/02-map.md` §3（邻接图）
- `docs/design/10-tech.md` §4.1（渲染分层）
- 半边数据结构（DCEL）：https://en.wikipedia.org/wiki/Doubly_connected_edge_list
- SVG path：https://developer.mozilla.org/en-US/docs/Web/SVG/Attribute/d

       function generateAnchors(p1, p2, isBoundary): Vec2[] {
         if (isBoundary) return [p1, p2]
         const seed = edgeHash(p1, p2)
         const rng = mulberry32(seed)
         const perp = perpUnit(p1, p2)  // 因为 p1<p2 排序后的法线，确定性
         const len = edgeLength(p1, p2)
         const result: Vec2[] = [p1]
         for (let j = 1; j < N_SUBDIV; j++) {
           const t = j / N_SUBDIV
           const along = lerp(p1, p2, t)
           const amplitude = Math.sin(t * Math.PI) * len * NOISE_RATIO
           const noise = (rng() - 0.5) * 2 * amplitude
           result.push([along[0] + perp[0] * noise, along[1] + perp[1] * noise])

         return result
       }

         const n = anchors.length
         const controls: Array<[Vec2, Vec2]> = []
         for (let i = 0; i < n - 1; i++) {
           const p0 = anchors[Math.max(0, i - 1)]
           const p1 = anchors[i]
           const p2 = anchors[i + 1]
           const p3 = anchors[Math.min(n - 1, i + 2)]
           const c1: Vec2 = [p1[0] + (p2[0] - p0[0]) / 6, p1[1] + (p2[1] - p0[1]) / 6]

           controls.push([c1, c2])
         }
         return controls
       }

         }

         function deriveAdjacency(sites: RawSite[]): Map<SiteId, SiteId[]> {
           const edgeToSites = new Map<EdgeId, SiteId[]>()
           for (const site of sites) {
             for (const ref of site.boundary) {
               const list = edgeToSites.get(ref.edge) ?? []
               list.push(site.id)
               edgeToSites.set(ref.edge, list)
             }
           }
           const adj = new Map<SiteId, Set<SiteId>>()
           for (const [, ids] of edgeToSites) {
             if (ids.length !== 2) continue
             const [a, b] = ids
             ;(adj.get(a) ?? adj.set(a, new Set()).get(a))!.add(b)
             ;(adj.get(b) ?? adj.set(b, new Set()).get(b))!.add(a)
           }
           return new Map([...adj].map(([k, v]) => [k, [...v]]))
         }

      export function buildSitePathFromBoundary(
        site: Site,
        edges: ReadonlyMap<EdgeId, MapEdge>,
      ): Path2D {
        const path = new Path2D()
        let isFirst = true
        for (const ref of site.boundary) {
          const edge = edges.get(ref.edge)
          if (!edge) continue
          const segments = getEdgeSegments(edge, ref.reverse)
          for (const seg of segments) {
            if (isFirst) {
              path.moveTo(seg.start[0], seg.start[1])
              isFirst = false
            }
            if (seg.kind === 'line') {
              path.lineTo(seg.end[0], seg.end[1])
            } else if (seg.kind === 'bezier') {
              path.bezierCurveTo(
                seg.c1[0], seg.c1[1],
                seg.c2[0], seg.c2[1],
                seg.end[0], seg.end[1],
              )
            }
          }
        }
        path.closePath()
        return path
      }

      function getEdgeSegments(edge: MapEdge, reverse: boolean): Segment[] {
        const anchors = reverse ? [...edge.anchors].reverse() : edge.anchors
        if (edge.curveType === 'polyline' || !edge.controls) {
          // 直线段
          return anchors.slice(1).map((end, i) => ({ kind: 'line' as const, start: anchors[i]!, end }))
        }
        // cubic-bezier：reverse 时不仅 anchors 反转，controls 也反转 + 每对 C1↔C2 swap
        const controls = reverse
          ? [...edge.controls].reverse().map(([c1, c2]) => [c2, c1] as readonly [Vec2, Vec2])
          : edge.controls
        return anchors.slice(1).map((end, i) => ({
          kind: 'bezier' as const,
          start: anchors[i]!,
          c1: controls[i]![0],
          c2: controls[i]![1],
          end,
        }))
      }
