## Context

### Original Request
M0 演示后用户反馈：
> NO，存在两个地图上的实现问题：
> 1. 相邻区块没有完全紧贴
> 2. 区块边界为不自然的折线，没有实现符合自然地图的完全曲线型边界随机生成
>
> 请考虑使用非实时渲染边界，而是提前生成好可填充的区域和渲染区块，在渲染时直接贴图的方案

### Diagnosis

**问题 1：相邻区块不紧贴**
- 根因：M0 T5 用了"方案 B"——`tools/generate-m0-map.ts` 内手工硬编码 5 组多边形，相邻多边形的顶点不共享，导致存在缝隙。
- 修复：必须用 **Voronoi 镶嵌** —— 数学上保证 cells 共享边。`d3-delaunay` 已在 devDependencies（M0 T5 原方案 A 未采用）。

**问题 2：边界为不自然折线**
- 根因：每 cell 仅 8-10 个 `lineTo` 直线段。
- 修复（双管齐下）：
  - **数据层**：Voronoi 共享边上细分 N=20 个点，沿法线方向施加 hash-seeded 平滑随机扰动；端点 falloff 确保 Voronoi 顶点不偏移（保证零缝隙）。
  - **渲染层**：`quadraticCurveTo` 通过中点的 C¹ 平滑曲线。

**用户的性能优化要求：预渲染 Tile Cache**
- 不在每帧实时计算/绘制曲线（开销大）
- 启动时一次性为每个 (site, faction) 组合预渲染到 OffscreenCanvas
- 运行时仅 `ctx.drawImage(cache[site][owner], 0, 0)` —— 极速

    1. 在 800×600 边界内随机生成 5 个种子点（用确定性 PRNG，seed=12345，便于复现）
    2. 应用 **Lloyd's relaxation** 4 次迭代（每次：计算 Voronoi → 用 cell 质心替换种子点 → 重新 tessellate）—— 让 cells 大小均匀
    3. 用 d3-delaunay 计算最终 Voronoi 镶嵌，clip 到 800×600 边界
    4. **Edge subdivision + hash-seeded perturbation**：
       - 对每个 Voronoi cell 收集所有边
       - 对每条边的两个端点排序（lexicographic），用 `hashEndpoints(p1, p2)` 计算 deterministic seed
       - 在边上等距插入 N=20 个中间点
       - 每个中间点 j（t = j/N，j ∈ [1, N-1]）：
         - `along` = lerp(p1, p2, t)
         - `noise` = (PRNG(seed + j).value - 0.5) × 2 × amplitude
         - `amplitude` = sin(t × π) × edgeLength × 0.08（端点 falloff）
         - `perpendicular` = perpDir(p1, p2) × noise
         - 最终点 = along + perpendicular
       - **关键**：相邻 cell 处理同一条 shared edge 时，hash 必须基于排序端点，确保两侧生成一致序列
    5. 推导邻接：Delaunay triangulation 的 neighbors 即 site adjacency
    6. 输出 `src/content/m0/sites.json`，符合 `M0DataSchema`：

    ```typescript
    import type { Site, Faction, SiteId, FactionId } from '@/shared/types'

    /** 预渲染的 (site, faction) tile —— 存为 HTMLCanvasElement */
    export type TileCache = ReadonlyMap<SiteId, ReadonlyMap<FactionId, HTMLCanvasElement>>

    const CANVAS_WIDTH = 800
    const CANVAS_HEIGHT = 600
    const BORDER_COLOR = '#1A1A1A'
    const TEXT_COLOR = '#666666'

    /** 在指定 canvas 上以平滑曲线填充 polygon + 描边 + 文字 */
    function paintSiteTile(
      canvas: HTMLCanvasElement,
      site: Site,
      color: string,
    ): void {
      const ctx = canvas.getContext('2d')
      if (!ctx) return
      ctx.clearRect(0, 0, CANVAS_WIDTH, CANVAS_HEIGHT)

      const path = buildSmoothPath(site.polygon)
      ctx.fillStyle = color
      ctx.fill(path)
      ctx.strokeStyle = BORDER_COLOR
      ctx.lineWidth = 1.2
      ctx.stroke(path)

      // 名称文字
      const [cx, cy] = centroid(site.polygon)
      ctx.fillStyle = TEXT_COLOR
      ctx.font = '14px sans-serif'
      ctx.textAlign = 'center'
      ctx.textBaseline = 'middle'
      ctx.fillText(site.name, cx, cy)
    }

    /** 通过中点的 C¹ quadratic Bezier 平滑曲线 */
    function buildSmoothPath(polygon: readonly (readonly [number, number])[]): Path2D {
      const path = new Path2D()
      const n = polygon.length
      if (n < 3) return path
      const mid = (i: number, j: number): [number, number] => [
        (polygon[i]![0] + polygon[j]![0]) / 2,
        (polygon[i]![1] + polygon[j]![1]) / 2,
      ]
      const start = mid(n - 1, 0)
      path.moveTo(start[0], start[1])
      for (let i = 0; i < n; i++) {
        const next = mid(i, (i + 1) % n)
        path.quadraticCurveTo(polygon[i]![0], polygon[i]![1], next[0], next[1])
      }
      path.closePath()
      return path
    }

    function centroid(polygon: readonly (readonly [number, number])[]): [number, number] {
      let cx = 0, cy = 0
      for (const [x, y] of polygon) { cx += x; cy += y }
      return [cx / polygon.length, cy / polygon.length]
    }

    /** 一次性构造所有 (site, faction) tile cache */
    export function buildTileCache(
      sites: ReadonlyMap<SiteId, Site>,
      factions: ReadonlyMap<FactionId, Faction>,
    ): TileCache {
      const cache = new Map<SiteId, Map<FactionId, HTMLCanvasElement>>()
      for (const [siteId, site] of sites) {
        const siteCache = new Map<FactionId, HTMLCanvasElement>()
        for (const [factionId, faction] of factions) {
          const canvas = document.createElement('canvas')
          canvas.width = CANVAS_WIDTH
          canvas.height = CANVAS_HEIGHT
          paintSiteTile(canvas, site, faction.color)
          siteCache.set(factionId, canvas)
        }
        cache.set(siteId, siteCache)
      }
      return cache
    }
    ```

    - 新的渲染逻辑：
      ```typescript
      const tileCache = useMemo(() => buildTileCache(sites, factions), [sites, factions])
      
      const draw = useCallback(() => {
        const canvas = canvasRef.current
        if (!canvas) return
        const ctx = canvas.getContext('2d')
        if (!ctx) return
        const now = performance.now()
        
        // 米色背景

        ctx.fillRect(0, 0, CANVAS_WIDTH, CANVAS_HEIGHT)
        
        // 对每个 site：贴图 cached tile（含 transition alpha-blend）
        for (const [siteId, site] of sites) {
          const siteTiles = tileCache.get(siteId)
          if (!siteTiles) continue
          
          const transition = transitionsRef.current.get(siteId)
          if (transition) {
            const elapsed = now - transition.startMs
            if (elapsed >= TRANSITION_DURATION_MS) {
              transitionsRef.current.delete(siteId)
              const tile = siteTiles.get(site.ownerId ?? '')
              if (tile) ctx.drawImage(tile, 0, 0)
            } else {
              const t = easeInOut(elapsed / TRANSITION_DURATION_MS)
              const fromTile = siteTiles.get(transition.fromFactionId)
              const toTile = siteTiles.get(transition.toFactionId)
              if (fromTile) {
                ctx.globalAlpha = 1 - t
                ctx.drawImage(fromTile, 0, 0)
              }
              if (toTile) {
                ctx.globalAlpha = t
                ctx.drawImage(toTile, 0, 0)
              }
              ctx.globalAlpha = 1
            }
          } else {
            const tile = siteTiles.get(site.ownerId ?? '')
            if (tile) ctx.drawImage(tile, 0, 0)
          }
        }
      }, [sites, factions, tileCache])
      ```
