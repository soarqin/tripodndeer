# 05 · 治国系统 · 外交 · 内政 · 经济 · 文化 · 谍报

> 战争只是峰值时刻。和平期的治国，决定战争的胜败。本文涵盖所有"非战"系统。

---

## 1. 设计目标

| 目标 | 含义 |
|------|------|
| **每个子系统都有"古中国味"** | 朝贡、合纵、变法、稷下、客卿——这些词应在游戏里活着 |
| **系统之间有耦合** | 内政影响经济，经济影响军事，文化影响外交，外交影响合法性…… |
| **决策不是数值题** | 政策选择应有性格，而非"看数值最高就选" |
| **节奏不喧宾夺主** | 治国系统服务于核心循环，而非扩成一个独立的城建游戏 |

## 2. 外交（Diplomacy）

### 2.1 外交关系基础维度

每对势力之间维护一份**关系档案**：

```yaml
relation:
  realm_a: realm_qin
  realm_b: realm_wei
  attitude: -45              # 态度 (-100 ~ +100)
  trust: 30                  # 信任度 (0–100)
  prestige_balance: -10      # 威望对比
  truce_until: 0             # 停战截止日（0 表示无停战）
  active_treaties: [non_aggression]
  shared_history: [war_362bc, alliance_320bc]
```

### 2.2 外交动作清单

| 动作 | 类型 | 触发后果 |
|------|------|---------|
| **使节问候** | 维护型 | 提升态度，缓慢 |
| **联姻** | 维护型 | 大幅提升信任，但可能有继承牵连 |
| **缔结盟约** | 同盟型 | 互相援助义务，违反代价大 |
| **互不侵犯** | 弱同盟 | 短期约束 |
| **军事同盟** | 强同盟 | 敌即敌、友即友 |
| **朝贡** | 不对等 | 弱者向强者岁币换和平、保留自治 |
| **称臣 / 内附** | 极不对等 | 成为附庸国 |
| **册封** | 仅天子 / 大国 | 赋予对方合法政体（如周王封秦伯） |
| **议和** | 战争结束 | 详见 [`04-systems-military.md`](./04-systems-military.md) |
| **宣战** | 起战 | 详见同上 |
| **羞辱 / 挑衅** | 攻击型 | 拉低对方威望、惹怒对方 |
| **交换土地** | 议和外延 | 玩家可主动提议 |
| **送质子** | 信任型 | 派王子作为人质换取互信 |
| **赎回质子 / 战俘** | 一次性 | 国库支付换人 |
| **退还合法性 / 转交合法性** | 政治型 | 大幅威望转移（罕见但戏剧性） |

### 2.3 合纵连横（Strategic Coalitions）

> 这是本作最具特色的外交系统。

**合纵（vertical）**：弱国联合抗强国
- 触发：当某国威望/势力达到 70%+ 区域霸权
- 玩家可作为发起者或被动加入
- 联盟期间共同对抗领头羊
- 一旦霸主削弱，联盟易瓦解

**连横（horizontal）**：强国分化弱国联盟
- 触发：玩家强大且有合纵威胁
- 通过"远交近攻"等机制各个击破
- 派遣纵横家是关键

#### 苏秦张仪式的事件链
- 当某位**辩士**（高交、高谋）服务玩家时
- 玩家可发动"合纵 / 连横"主张
- 触发跨国事件链，影响多势力关系

### 2.4 朝贡体系（Tributary System）

中国古代外交的核心。本作以此替代欧洲式的"附庸"概念：

| 关系 | 性质 | 玩法 |
|------|------|------|
| **天子-诸侯** | 名义臣属（春秋初期普遍） | 诸侯须岁朝、纳贡，天子可册封 |
| **盟主-盟国** | 春秋霸业 | 盟主主持会盟、惩罚不来朝者 |
| **大国-属国** | 战国强弱 | 属国保自治但需称臣纳贡 |
| **中原-边夷** | 文化中心-边缘 | 仅形式上的朝贡 |

朝贡关系破裂会触发**讨伐借口**（详见军事文档）。

### 2.5 礼乐征伐自天子出（合法性传递）

- 周天子拥有**册封权**
- 诸侯可向天子求封爵位（公→侯→伯→子→男 反向：周末年很多侯僭称王）
- 在春秋早期，没有天子认可的征伐属"无名之师"
- 战国后这套机制崩溃，但玩周天子时仍是核心玩法

## 3. 内政（Internal Governance）

### 3.1 政令与变法（Edicts & Reforms）

**政令（Edict）**：可频繁颁布的中短期政策
- 例：「劝农令」（农业 +20%，3 年）、「征兵令」（兵源 +30%，但稳定 -10）
- 玩家可同时持有 2–3 条政令
- 颁布与撤销有冷却

**变法（Reform）**：罕见的**结构性改革**，不可逆
- 例：商鞅变法、吴起变法、赵武灵王胡服骑射
- 触发条件：需要相应**革新者**人才 + 君主支持 + 政治稳定窗口
- 触发即进入**多年事件链**：阻力、流血、成功/失败的分支
- 成功则获得永久重大加成；失败则反弹（人才被诛、合法性大损）

> 变法是单局游戏的关键节点。设计时确保它的**戏剧性**与**风险性**。

### 3.2 户籍与人口

```yaml
province:
  population: 250000          # 总人口
  population_growth: 0.4%/yr  # 受年景、稳定性、政策影响
  registered_households: ...  # 编户齐民数（影响税收能力）
  manpower_capacity: 25000    # 可征兵上限
  loyalty: 75                 # 忠诚（影响治理效率、起义风险）
  cultural_identity: chinese_qin
  development_level: 4        # 开发度（投入资源升级）
```

### 3.3 太守 / 郡守

- 每郡可任命一位太守（详见 [`03-factions.md`](./03-factions.md)）
- 优秀太守对该郡：
  - 税收 + / 农业 +
  - 人口增长 +
  - 忠诚维持高
  - 战时提供地方动员加成
- 暴政或弱能太守反之

### 3.4 政体改革路径

- 春秋初期：分封制（诸侯有大量封地、世袭卿大夫）
- 春秋中后期：六卿专权（世袭逐渐架空君权）
- 战国：郡县制改革（君主直辖太守，世袭瓦解）
- 战国后期：法家集权制（秦最彻底）

玩家通过变法可主动推进。**不变法**则慢慢被时代抛弃，但保留分封制有合法性优势。

## 4. 经济（Economy）

### 4.1 资源体系

| 资源 | 来源 | 用途 |
|------|------|------|
| **粮（粟）** | 农业 | 维系军队、人口、税收 |
| **铜钱（货币）** | 商税、矿冶 | 国库、贸易、外交 |
| **铜、铁** | 矿场 | 兵器、农具 |
| **盐、铁专营** | 战国后期改革 | 巨大财政来源 |
| **战马** | 草场（北、西） | 骑兵专属 |
| **丝麻** | 桑麻产区 | 兵器装备、贸易 |
| **木材** | 林地 | 战车、攻城器、舰船 |
| **玉、丹砂** | 特殊产地 | 礼制、外交礼物 |

### 4.2 经济结算频率

- **月度**：粮草、税收、军饷
- **季度**：人口结算、灾害检查
- **年度**：贸易总账、外交岁贡

### 4.3 灾害与丰歉

| 事件 | 影响 |
|------|------|
| **丰年** | 农业 +30%，人口增长加速 |
| **歉年** | 农业 -25%，可能流民 |
| **大旱 / 大水** | 局部毁灭性，需赈灾 |
| **蝗灾** | 区域性大减产 |
| **瘟疫** | 人口骤降，军队尤其受打击 |

灾害发生时玩家须决定：
- 开仓赈灾（消耗粮草，民望 +）
- 减税（财政损失，民望 +）
- 强征（短期纾解，民望 −）
- 不管（民望 −−，可能起义）

### 4.4 贸易与商路

- 邑之间可建立贸易路线
- 跨势力贸易需对方允许
- **西域、海上、巴蜀三大潜在商路**（视时代）
- 战争自动切断敌对国之间的贸易

## 5. 文化与意识形态（Culture & Ideology）

### 5.1 文化的两层模型

每个邑/郡有：
- **文化标签**（chinese_qin / chinese_chu / yi_dong / di_xirong / ...）
- **文化认同强度**（0–100）

文化标签**慢变**（数十年甚至上百年才会改变），认同强度**快变**（受统治、教育、屠戮影响）。

### 5.2 文化转化

```
被外族征服 → 短期: 认同强度 ↓ (痛恨新主)
              中期: 设学官、移民、通婚 → 认同强度恢复
              长期: 文化标签可能转变
```

> 历史上"楚虽三户，亡秦必楚"——被征服了不一定就同化。这是叛乱的源头。

### 5.3 意识形态系统（Ideology）

每个势力有**意识形态倾斜面板**（见 [`03-factions.md`](./03-factions.md) §2.4）。倾斜由以下决定：
- 君主的个人倾向
- 主政人才的流派
- 政令与变法历史
- 学宫的派系

不同倾向解锁不同**学派建筑**和**特殊政令**。

#### 学宫系统

- 在某些大都会可建造**学宫**（齐有稷下、魏有西河等）
- 学宫每年产出人才（按势力意识形态倾向产出对应流派人才）
- 学宫可作为**意识形态输出基地**（影响他国学子流向）

### 5.4 礼乐与威望

- **威望（Prestige）**：势力的国际地位
- 来源：合法性、文化影响、武功、礼制守护、举办盟会
- 用途：外交筹码、册封他国的能力、文化吸引力

## 6. 谍报（Intelligence & Subterfuge）

### 6.1 情报覆盖度

每对势力之间维护**情报覆盖度**（0–100）：
- 影响玩家在地图上能看到对方多少信息（驻军、军令、内部）
- 通过**派遣间者**、**通商**、**联姻**等提升

### 6.2 谍报动作

| 动作 | 作用 | 风险 |
|------|------|------|
| **侦察** | 获取对方军事情报 | 低 |
| **散布谣言** | 降低对方稳定 | 中 |
| **离间** | 降低敌方人才忠诚 | 中 |
| **策反郡守** | 让某郡叛归我方 | 高 |
| **策反将领** | 让敌将临阵倒戈 | 极高 |
| **暗杀人才** | 使敌方损失关键人才 | 极高 |
| **暗杀君主** | 使敌国陷入继承危机 | 极高（败露=外交灾难） |
| **窃取战法 / 文化** | 获取对方独有内容 | 中 |

### 6.3 间者（Spy）

- 可派出独立活动的间者人才
- 长期潜伏获得稳定情报
- 可被对方反间（事件触发）

### 6.4 反谍（Counter-Intelligence）

- 御史 / 司徒等岗位可提升反谍效率
- 高谋君主主政时反谍更强
- 失败的间谍/刺杀任务**必须暴露**给玩家（让被谋杀方有可识破的机会）

## 7. 内部稳定（Internal Stability）

### 7.1 稳定性数值

```yaml
realm:
  stability: 60       # (0–100)
```

低稳定性触发：
- 起义 / 反叛
- 派系内斗
- 流民
- 太守怠工
- 经济效率折损

### 7.2 稳定性影响因素

| 增加 | 减少 |
|------|------|
| 君主威望 + 民望 | 暴政、连战、加税 |
| 优秀太守 | 君主薨逝（动荡期） |
| 文化认同高 | 屠城、强征、瘟疫 |
| 仁政 / 大赦 | 派系冲突 |

### 7.3 派系（Factions Within Realm）

每个势力有内部派系：
- **君党**：忠君派
- **公族 / 宗室**：王族成员
- **世族 / 卿族**：世袭士大夫
- **军功派**：因战功崛起的新贵
- **改革派 / 守旧派**：变法时期的对立
- **外姓 / 客卿**：异国来的人才

派系平衡 → 政局稳定
派系失衡 → 政变、分裂、颠覆（如田氏代齐、三家分晋）

> 这一系统让"内部斗争"和"外部战争"具有同等戏剧张力。

## 8. 事件系统（Event System）

### 8.1 事件类型

| 类型 | 触发 | 例子 |
|------|------|------|
| **历史事件** | 时间+条件 | 三家分晋、田氏代齐、商鞅变法 |
| **角色事件** | 与人才相关 | 商鞅来投、吴起被谗、白起拒赵 |
| **危机事件** | 状态触发 | 灾荒、瘟疫、内乱、异族入侵 |
| **机遇事件** | 状态触发 | 名相主动来投、邻国主动求盟 |
| **随机事件** | 随机+权重 | 边境冲突、流民事件、献宝 |
| **决战事件** | 大战节点 | 长平之战、马陵之战式的大型战役叙事 |

### 8.2 事件的呈现

每个事件都有：
- **古典叙事文本**（不要 AI 工具感）
- 1–4 个**选项**
- 每个选项**预览后果**（不全揭示，留 1–2 个隐藏后果）
- 可点击进入相关界面（势力、人物、地图）

### 8.3 数据驱动

事件存为数据文件（JSON/YAML），引擎查表执行。详见 [`06-scenarios.md`](./06-scenarios.md)。

## 9. 系统耦合矩阵（System Coupling Matrix）

> 这张表用来检查系统之间是否真的"互相影响"。如果一行/列都是 ❌，说明那个系统是孤立的，需要重新设计。

|  | 军事 | 外交 | 内政 | 经济 | 文化 | 谍报 |
|--|------|------|------|------|------|------|
| **军事** | — | ✅ 战争借口/议和 | ✅ 战疲影响稳定 | ✅ 军费 | ✅ 军功爵改文化 | ✅ 战时谍报激增 |
| **外交** | ✅ 同盟参战 | — | ✅ 政体影响信任 | ✅ 通商 | ✅ 文化亲缘 | ✅ 间者促和 |
| **内政** | ✅ 兵源 | ✅ 改革变外交 | — | ✅ 太守治产 | ✅ 政令影响文化 | ✅ 反谍 |
| **经济** | ✅ 军费 | ✅ 朝贡 | ✅ 税收基础 | — | ✅ 商旅传文化 | ✅ 商人间谍 |
| **文化** | ✅ 士气 | ✅ 文化软实力 | ✅ 学宫产人才 | ✅ 文化区贸易 | — | ✅ 文化间者 |
| **谍报** | ✅ 战前情报 | ✅ 谍报泄漏致外交灾难 | ✅ 内部稳定 | ✅ 商业间谍 | ✅ 文化窃取 | — |

✅ 表示存在设计上的耦合点。所有交叉点都应在实现时被覆盖。

| # | 问题 | 备选 |
|---|------|------|
| OQ-1 | 资源种类是否过多？ | 推荐 MVP：粮、钱、马、铁、人口；其余作为修正符号；日后扩展 |
