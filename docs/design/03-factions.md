# 03 · 势力 · 君主 · 人才

> 玩家不是在统治抽象的色块，而是在做一个**有血有肉的君主**。

---

## 1. 设计目标

| 目标 | 含义 |
|------|------|
| **势力差异化** | 玩楚和玩秦应有完全不同的体验，不是数值差，是**玩法差** |
| **君主有命数** | 君主会老、会病、会死，玩家面对继承的现实张力 |
| **人物有故事** | 每个人才不是功能模块，是有性格、有抱负、有结局的人 |
| **不喧宾夺主** | 人物系统服务于战略，不应让玩家陷入"养成手游"的微观经营 |

---

## 2. 势力（Realm）

### 2.1 势力的核心定义

```yaml
realm:
  id: realm_qin
  name: 秦
  full_title: "秦国"
  ruling_house: 嬴             # 国姓
  capital: site_xianyang
  legitimacy: 65               # 合法性 (0–100)
  prestige: 45                 # 威望 (0–100)
  state_form: kingdom          # 政体类型（见 §2.3）
  cultural: chinese_qin        # 主体文化
  ideology_lean: legalism      # 意识形态倾向（见 §2.4）
  treasury: 8000               # 国库
  manpower: 50000              # 兵源池
  ruler: char_xxx              # 当前君主 ID
  vassals: []                  # 附庸
  liege: realm_zhou            # 名义宗主（春秋初期为周）
  traits: [shang_yang_reform_pending, frontier_warrior]
```

### 2.2 势力的差异化（Factional Identity）

每个势力应有 **2–4 个独特特性（Realm Traits）**，赋予独特玩法体验。

#### 春秋战国首发示例

| 势力 | 独特特性 | 玩法影响 |
|------|---------|---------|
| **秦** | · 商鞅变法路径<br>· 西陲蛮夷之名<br>· 函谷天险 | 可走极端军国主义路线；外交上其他国家更难信任；地理上易守 |
| **楚** | · 地大兵众<br>· 蛮荒未化<br>· 王室宗法松散 | 起步军力强但行动迟缓；文化扩张慢；君权易被分食 |
| **齐** | · 鱼盐之利<br>· 稷下学宫<br>· 王室有外姓干政（田氏） | 经济富甲诸侯；可独占文化爆点；可能爆发"田代姜齐" |
| **晋（前期）** | · 六卿专权<br>· 中原门户<br>· 强军传统 | 一开局就在"内部分裂"危机倒计时（→三家分晋） |
| **韩** | · 兵器精良（弩弓）<br>· 处四战之地 | 单兵装备好，但地理位置最差，游戏开始就被围 |
| **赵** | · 胡服骑射潜力<br>· 北接草原 | 改革后可解锁强骑兵；需平衡北方异族 |
| **魏** | · 文侯遗风<br>· 河西要地 | 早期已是中原最强，但守成压力大；变法窗口期短 |
| **周（玩家可选 / AI）** | · 天子合法性<br>· 礼乐宗法 | 军事最弱但**外交合法性顶级**；玩法极度特殊（详见后） |
| **燕** | · 偏远北疆<br>· 苦寒善守 | 起步弱但安全；适合慢热长跑流派 |

> **设计原则**：每个势力的特性应直接影响玩家的**战略思路**（不止是数值修正）。

#### 周天子作为特殊势力

周王室是本作可玩的最有趣势力之一：
- 军事最弱（三百乘兵车 vs 大国数千乘）
- 但拥有独占的"册封"权——可以将自己的合法性转借诸侯
- 玩法变成"仲裁者"：通过册封、调停来制衡列强
- 胜利路径独特：不靠军事征服，而是**复兴礼乐**

> 这本身就是一种"反向战略"，让懂历史的玩家跃跃欲试。

### 2.3 政体形态（State Form）

| 政体 | 含义 | 修正 |
|------|------|------|
| **天子（King of Kings）** | 名义天下共主（如周天子） | 合法性极高，但实力受限 |
| **王国（Kingdom）** | 称王的诸侯（春秋后期/战国） | 合法性高、外交盟交易筹码强 |
| **公国（Duchy）** | 春秋初期的封建诸侯 | 合法性中、需向天子表忠 |
| **僭主（Usurper）** | 通过政变上位的政权（如田氏代齐前期） | 合法性低、易内乱 |
| **部族联盟（Tribal Confederacy）** | 边境异族（义渠戎、林胡等） | 不参与中原"礼制"系统、走另一套规则 |

政体的转换需触发条件（如周王册封、巨大威望积累、政变事件链）。

### 2.4 意识形态倾向（Ideological Lean）

势力可在以下意识形态间倾斜（每方向独立，可混合）：

| 倾向 | 古典对应 | 政策风格 | 加成方向 |
|------|---------|---------|---------|
| **法家** | 商鞅、韩非、李斯 | 严刑峻法、奖励耕战 | 军事+、内政效率+、人心/外交− |
| **儒家** | 孔孟之道 | 礼乐教化、宗法秩序 | 合法性+、稳定+、改革慢 |
| **道家** | 黄老之学 | 无为而治、与民休息 | 经济恢复+、扩张意愿− |
| **墨家** | 墨子 | 兼爱非攻、技术工艺 | 军工+、防御工事+、扩张代价+ |
| **纵横家** | 苏秦、张仪 | 合纵连横、外交博弈 | 外交+、谍报+、信任修复慢 |
| **兵家** | 孙武、吴起 | 重兵权、谋战术 | 战法解锁、将领效率+ |

详见 [`05-systems-statecraft.md`](./05-systems-statecraft.md) 文化与意识形态章节。

## 3. 君主（Ruler）

### 3.1 君主的属性

```yaml
ruler:
  id: char_qin_xiaogong
  name: 嬴渠梁
  title: 秦孝公
  dynasty: 嬴
  birth_year: -381
  death_year: null            # 未定（依寿命/事件）
  age: 22                     # 当前年龄
  health: 75                  # 健康值
  attributes:
    martial: 12               # 武 (0–20)
    administration: 11        # 政
    diplomacy: 8              # 交
    intrigue: 6               # 谋
    learning: 13              # 学
    charisma: 14              # 魄（影响人才招揽）
  traits: [reformist, ambitious, patient]
  personality: builder        # 性格原型（见 §3.3）
  legitimacy_personal: 80     # 个人合法性
  popularity: 65              # 民望
  spouse: char_xxx
  heir: char_xxx              # 储君
  children: [char_yyy, char_zzz]
```

### 3.2 君主的属性维度

借鉴中国古代评价君主的传统维度：**文（学/政）、武（武）、谋（智/谋）、德（仁/魄）**。我们用 5 项数值：

| 属性 | 古典对应 | 影响 |
|------|---------|------|
| **武** | 武略/统兵 | 亲征加成、武将忠诚、战法解锁 |
| **政** | 政事/吏治 | 内政效率、税收、政令通过率 |
| **交** | 外交/纵横 | 外交成功率、盟约稳定 |
| **谋** | 智略/权术 | 谍报、识破阴谋、平叛 |
| **学** | 学问/文化 | 教化、文化输出、人才识别 |
| **魄** | 魅力/威望 | 人才招揽、民望、合法性自然增长 |

### 3.3 君主性格原型（Personality Archetypes）

每位君主有 1 个**性格原型**（用于 AI 决策、事件触发）和若干**特性 traits**（修正数值）：

| 原型 | 倾向 | 例子 |
|------|------|------|
| **创业者（Builder）** | 改革、扩张、不畏险 | 秦孝公、楚悼王 |
| **守成者（Steward）** | 稳定、休养、避战 | 文景之治型 |
| **征服者（Conqueror）** | 扩张、强硬、冒险 | 秦昭襄王、赵武灵王 |
| **谋略家（Schemer）** | 阴谋、间谍、远交近攻 | 春申君型 |
| **暴君（Tyrant）** | 高压、独裁、消耗合法性 | 桀纣型 |
| **昏庸（Incompetent）** | 决策迟疑、易被操控 | 末代昏君 |
| **仁君（Benevolent）** | 重民、合法性自然增长 | 三代圣王理想型 |
| **儒雅（Learned）** | 重文、慢战 | 宋襄公型 |

### 3.4 君主的命数（Mortality）

- 君主有自然寿命（基础值 60–70，受健康、医疗、纵欲事件影响）
- 健康值会随年龄、事件递减
- 君主薨逝触发**继承事件**（详见 §3.5）

### 3.5 继承（Succession）

```
┌────────────────┐
│ 君主薨逝事件   │
└───────┬────────┘
        ▼
┌────────────────────────┐
│ 检查继承法 + 储君人选  │
└───────┬────────────────┘
        ▼
   有合法储君？
        │
   ┌────┴────┐
   是        否
   │         │
   ▼         ▼
平稳继承  继承危机
              │
              ├─ 兄终弟及之争
              ├─ 庶子之争
              ├─ 权臣干政
              └─ 内乱 / 分裂
```

**继承法类型**：
- **嫡长子继承制**（春秋以后主流）
- **兄终弟及**（商代、部分诸侯）
- **择贤而立**（极少数特殊事件）
- **由权臣推立**（衰世）

> 继承危机是高潮事件，必须自动暂停（[`01-core-loop.md`](./01-core-loop.md) §3.2）。

## 4. 人才系统（Talent System）

### 4.1 设计原则

> **人才是战略资源，不是英雄养成对象**

- 人才数量克制（每势力同时只有 5–15 位重要人物）
- 不做装备、技能树、经验值
- 重在**任用**和**事件**，而非"培养"

### 4.2 人才的核心属性

```yaml
character:
  id: char_shang_yang
  name: 公孙鞅
  alias: 商鞅
  birth_year: -390
  death_year: -338            # 历史死亡时间，可能因玩家行为改变
  origin_realm: realm_wei     # 出身势力
  current_employer: realm_qin # 当前服务势力
  attributes:
    martial: 8
    administration: 18        # 极强
    diplomacy: 10
    intrigue: 14
    learning: 17              # 极强
    charisma: 9
  loyalty: 85
  ambition: high
  ideology: legalism          # 思想流派
  specialty: reformer         # 专长（见 §4.4）
  traits: [reformer, pragmatic, ruthless]
  posts: [chancellor]         # 任职
```

### 4.3 人才的来源

| 来源 | 触发条件 |
|------|---------|
| **历史出生** | 剧本设定的历史人物在指定年份"现身" |
| **野生人才** | 引擎随机生成的小人物（按时代风格命名） |
| **门客投奔** | 君主魅力高、势力开明时主动来投 |
| **敌国流亡** | 敌国失意人才主动求见（如商鞅离魏入秦） |
| **学宫毕业** | 拥有学宫建筑可周期性产出人才 |
| **战俘归降** | 战胜后有几率招降敌国将领 |

> 历史人物**默认按史实在合理时间出现**，但玩家行为可改变其去向（齐留商鞅、楚得吴起的"假如"）。

### 4.4 人才的专长（Specialty）

| 专长 | 适合岗位 | 加成 |
|------|---------|------|
| **统帅（General）** | 军团统帅 | 大兵团作战 |
| **猛将（Warrior）** | 先锋、副将 | 单兵战法、斩将 |
| **谋士（Strategist）** | 军师、参谋 | 战术建议、识破计谋 |
| **吏才（Administrator）** | 太守、相 | 内政、税收、人口 |
| **革新者（Reformer）** | 变法主官 | 解锁/加速改革 |
| **辩士（Diplomat）** | 使节、外相 | 外交效率、合纵连横 |
| **间者（Spy）** | 谍报、刺杀 | 情报、暗杀 |
| **学者（Scholar）** | 学官、教育 | 文化扩散、人才教育 |
| **工师（Engineer）** | 工程营造 | 城防、攻城器械、水利 |

一个人才可有 1–2 个专长。

### 4.5 任用（Appointment）

| 岗位 | 数量 | 加成 |
|------|------|------|
| **相 / 丞相** | 1 | 全国内政效率 +X% |
| **大将军** | 1 | 全军军事加成、战法解锁 |
| **太尉 / 上将军** | 1 | 兵源池、训练效率 |
| **御史 / 司徒** | 1 | 谍报、识破阴谋 |
| **使节** | 多 | 派遣外交任务的执行人 |
| **太守 / 郡守** | 等于郡数 | 该郡的内政、文化、稳定 |
| **军团长** | 等于军团数 | 该军团的战斗力 |
| **门客 / 食客** | 多 | 储备池，未任职 |

任命与免职是核心玩家动作。任命错误（让间谍当御史）后果严重；任命英才能力错配（让纵横家去守城）则浪费。

### 4.6 忠诚与离心

人才有忠诚值（loyalty 0–100）。低于阈值会触发：
- **怠工**：能力效率打折
- **求去**：主动辞职
- **私通**：暗中与敌国联系
- **叛变**：举郡叛乱、出卖情报、刺杀君主

**忠诚的来源**：
- 君主魅力 / 君臣相得
- 同流派意识形态
- 受重用程度
- 经济激励（俸禄、封地）
- 个人野心是否被满足
- 君主是否英明（昏君下能臣易离心）

**降低忠诚的因素**：
- 君主猜忌（主动监视）
- 长期不被任用
- 受到屈辱
- 家国仇怨（祖籍国被本国攻打）
- 敌国间谍策反

## 5. 历史人物的"史实回响"

> 这是历史还原的关键设计点。

### 5.1 时间窗口（Time Window）

每位历史人物有：
- **出生窗口**：早于此年不会出现
- **历史寿命**：参考史实
- **关键履历事件**：如"商鞅 35 岁前后将出走"，"吴起在魏国受谗后将奔楚"

### 5.2 史实分支（Historical Branches）

引擎在历史人物的**关键节点**会触发事件：

> 「公孙鞅在魏久不得志，欲离魏。」
> [玩家若为魏：是否挽留？]
> [玩家若为秦：是否派使者招揽？]
> [玩家若为他国：是否插手？]

玩家的选择会改变这位人物的命运，但**他依然是商鞅**——他到了哪个国家就去推动那个国家的变法。这就是「形似神在」。

### 5.3 历史人物的"灵感天赋"

历史人物拥有可解锁的**特殊技能**（仅历史人物可有）：

| 人物 | 技能 |
|------|------|
| **商鞅** | 「徙木立信」——首次颁布严法时全国稳定 +20，3 年内政令效率 +30% |
| **吴起** | 「率先士卒」——亲临前线时战斗力 +25%，但伤亡风险高 |
| **苏秦** | 「合纵长策」——可发起多国联盟事件 |
| **白起** | 「人屠」——歼灭战胜利时彻底削弱敌国兵源池，但威望反弹 |
| **管仲** | 「相国之政」——内政效率 +50%，齐有此人则成霸业事件解锁 |

> 这些技能是叙事彩蛋，让玩家"用对人"产生历史性的高光时刻。

## 6. 玩家与角色的关系

玩家**扮演的是势力本身**，而不是任何具体君主。这意味着：

- 君主死亡 ≠ 游戏结束（除非无人继承）
- 玩家可"穿越"多代君主
- 但玩家受**当前君主性格**影响——昏君任内某些选项不可用、合法性下降影响外交筹码

> 这是 Paradox 的 EU/HOI 路线（玩国），不是 CK 路线（玩人）。但 CK 的人物深度我们尽量保留。

## 7. 待确认问题（Open Questions）

| # | 问题 | 备选 |
|---|------|------|
| OQ-1 | 是否实现完整的家族 / 血缘 / 联姻系统？ | (a) 简化为父子+王后（推荐 MVP）（b) 全家族建模（CK 风） |
| OQ-2 | 人才数量上限怎么定？ | 推荐：每势力同时活跃 ≤15 位重要人物，避免微观经营负担 |
| OQ-3 | 玩家死亡 / 亡国后是否允许"接管继承人"或"切换势力"？ | 推荐：仅"亡国时给予观战或读档"，不允许切换势力 |
| OQ-4 | 春秋时期是否仍区分"王族"与"卿大夫"？例如晋的六卿？ | 推荐：建模为"势力内的次级派系"，影响内部稳定 |
| OQ-5 | 人才生成时其姓名风格是否区分时代？（春秋有姓+氏，战国渐合一） | 推荐：是。剧本数据中提供时代命名规则 |
