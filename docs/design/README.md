# 鼎鹿 · Tripod & Deer

> 「**问鼎中原，逐鹿天下**」——一款以中国古代为舞台、可暂停即时制大战略游戏。

---

## 一页纸总览（One-Pager）

| 维度 | 决策 |
|------|------|
| **核心体验** | 让玩家亲历一场「群雄并起 → 折冲樽俎 → 版图归一」的古典叙事 |
| **类型** | 单机大战略（Grand Strategy）· 可暂停即时制（Pausable Real-Time）· 地图涂色（Map Painting） |
| **参考坐标** | Paradox 的 *Europa Universalis* / *Crusader Kings*，叠加中国古代题材 |
| **首版面向** | 单人 vs AI；本地存档读档 |
| **首发剧本** | **春秋战国**（公元前 453 年三家分晋为起点，至秦灭六国） |
| **多时代架构** | 数据驱动剧本，未来可扩展 *三国* / *五代十国* / *元末群雄* 等 |
| **历史观** | **形似神在**——地理、势力、文化符号尊重史实；玩法机制服务于乐趣 |
| **典型对局时长** | 完整一统约 3–5 小时（含暂停决策时间）；可随时存档退出 |
| **网页技术栈（建议）** | TypeScript + Vite + React UI + Canvas 地图渲染 + IndexedDB 存档 |
| **反目标（Anti-Goals）** | 不是回合制三国志、不是 Risk、不是即时战术（RTS）、不强求碎片时间 |

---

## 设计支柱（Design Pillars）

1. **历史的重量**：每一座城、每一道关、每一位君主，都应让懂历史的玩家会心一笑。
2. **决策的尊严**：暂停是设计语言。每个不可逆决策，都值得玩家停下来思考。
3. **地图的史诗**：地图是主角。涂色不是占格子，而是文明气运的延伸与收缩。

详见 [`00-vision.md`](./00-vision.md)。

## 术语速查（Glossary）

| 术语 | 含义 |
|------|------|
| **Tick / 时步** | 模拟时钟的最小推进单位（建议 = 1 旬，即 10 天） |
| **Region / 邑** | 地图上最小的领土归属单元 |
| **Province / 郡** | 由若干邑构成的中等行政单元 |
| **Realm / 势力** | 一个独立政权（齐、楚、晋、秦……） |
| **Liege / 宗主** | 拥有附庸势力的上级政权（如周天子） |
| **Vassal / 附庸** | 名义臣属的下级势力 |
| **Pause-Decision Loop / 暂停决策循环** | 本作核心节奏：观察 → 暂停 → 决策 → 推进 → 反馈 |
| **Map Painting / 地图涂色** | 通过领土归属变化实时更新地图色块的视觉反馈 |
| **形似神在** | 历史还原原则：外形（地理/人物/符号）尊重史实，神髓（玩法/数值）服务体验 |
