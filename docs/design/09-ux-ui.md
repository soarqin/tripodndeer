# 09 · 界面与交互设计（UX/UI）

> 本作 90% 的玩家时间盯着同一块屏幕。这块屏幕设计得好，游戏就成功一半。

---

## 1. 设计目标

| 目标 | 含义 |
|------|------|
| **地图为主，UI 为辅** | UI 是地图的"叠加层"，绝不喧宾夺主 |
| **信息分层** | 默认极简，按需展开。不让新手看到第一眼就劝退 |
| **操作一致** | 鼠标右键、键盘热键、上下文菜单等全游戏统一 |
| **古典视觉** | 不是"做古装的现代游戏"，而是"古朴中带着信息密度的工具" |
| **无障碍** | 色盲友好、字号可调、键盘可全控 |

## 2. 主屏幕（Main Screen）布局

```
┌──────────────────────────────────────────────────────────────────┐
│  [顶栏] 国号 / 君主 / 国库 / 兵源 / 民望 / 威望 / 时间 / 速度    │
├────────────┬─────────────────────────────────────┬──────────────┤
│            │                                     │              │
│            │                                     │              │
│  [左栏]    │           [主地图 · 居中]            │  [右栏]      │
│  事件通知  │     缩放级别决定信息密度              │  胜利进度    │
│  日志摘要  │     悬停高亮 / 点击聚焦              │  胜利里程碑  │
│  待办提醒  │     动态军团图标                     │  小地图     │
│            │                                     │              │
│            │                                     │              │
├────────────┴─────────────────────────────────────┴──────────────┤
│ [底栏] 主菜单(王宫) | 军事 | 外交 | 内政 | 文化 | 谍报 | 人物    │
│       时间控制条 [⏸ ▶ ▶▶ ▶▶▶ ▶▶▶▶ ▶▶▶▶▶]      地图模式切换    │
└──────────────────────────────────────────────────────────────────┘
```

### 2.1 顶栏（Top Bar）

**关键资源条**——永远可见，让玩家随时知道势力状态。

| 元素 | 内容 |
|------|------|
| 国号 + 君主像 | 点击进入"王宫"主面板 |
| 国库（💰） | 当前数 + 月度收支预测 |
| 兵源池（🛡） | 可征兵数 / 上限 |
| 民望（民） | 0–100 进度条 |
| 威望（🏛） | 0–100 进度条 + 当前国际地位 |
| 稳定性（⚖） | 0–100 |
| 当前日期 | 年 / 月 / 旬 |
| 时间速度 | 当前档位（暂停/1x/2x...） |

> 鼠标悬停每项 → tooltip 展开**该数值的影响来源**（如"民望 65：基础 50 + 太守加成 +10 + 仁政令 +5"）。

### 2.2 主地图（Center Map）

详见 [`02-map.md`](./02-map.md)。

**关键交互**：
- 鼠标拖动平移、滚轮缩放
- 单击邑 → 选中并显示该邑详情面板
- 右键邑 → 上下文菜单（驻军、迁徙、建造、外交…）
- 双击 → 聚焦缩放
- 框选 → 多选军团

### 2.3 左栏（Notification Sidebar）

可折叠的通知与日志栏：

- **顶部**：紧急通知（L1/L2 级）
- **中部**：可滚动的事件日志（L3 级累积）
- **底部**：待办提醒（"君主即位 5 年仍未册封储君"）

### 2.4 右栏（Progress Sidebar）

可折叠：

- 胜利路径进度条（5 条同时显示）
- 最近里程碑
- 小地图（mini-map）：战略级缩略

### 2.5 底栏（Action Bar）

主要功能入口 + 时间控制：

- **王宫**：主面板，势力总览
- **军事**：军团列表、将领、战法
- **外交**：势力关系、盟约、使节
- **内政**：政令、变法、太守
- **经济**：财政、贸易、灾害
- **文化**：意识形态、学宫、文化扩散
- **谍报**：间者、情报覆盖
- **人才**：人才库、招募、任免
- **时间控制条**：1 个暂停 + 5 档速度

> 数字快捷键 1–8 对应主面板，空格暂停，+/- 调速。

## 3. 信息密度的分层（Information Density Layers）

### 3.1 三级展开原则

| 层级 | 触发 | 信息量 |
|------|------|--------|
| **零层（Glance）** | 不操作，只看 | 顶栏数字、地图涂色、通知图标 |
| **一层（Hover）** | 鼠标悬停 | 数字解释、邑名、势力名 |
| **二层（Click）** | 点击 | 详细面板、子菜单 |
| **三层（Drill）** | 进入子面板 | 完整数据、历史曲线、依赖关系 |

> 任何一项数据，玩家应能在 ≤3 次点击内追溯到它的**全部来源**。

### 3.2 详细面板（Detail Panels）的统一结构

每个面板（势力、邑、军团、人物、外交关系...）应遵循统一布局：

```
┌─ 头部：标题 + 关键徽标（颜色、势力符号）─┐
├─ 概览：3–5 个最重要数值 ────────────────┤
├─ Tab 1: 详情 ───────────────────────────┤
├─ Tab 2: 历史 ───────────────────────────┤
├─ Tab 3: 行动 ───────────────────────────┤
└─ 底部：常用动作按钮 ─────────────────────┘
```

## 4. 关键界面（Key Screens）

### 4.1 王宫主面板（Realm Overview）

玩家点击国号或按 1 键进入。一目了然你的势力状态：

- 君主信息（健康、特性、即位年限、储君）
- 国力总览（领土、人口、国库、兵力、威望）
- 当前危机预警
- 当前已颁政令、变法状态
- 主政人才列表
- 国教 / 意识形态分布

### 4.2 军事面板

- 所有军团位置 + 状态
- 将领列表（含未任职将领）
- 战法解锁进度
- 兵源池趋势图（近 5 年）
- 当前战争状态（如有）

### 4.3 外交面板

**核心控件**：势力关系矩阵
- 横轴：所有已知势力
- 纵轴：本国、关系类型
- 单元格颜色：绿（盟）/ 黄（中立）/ 红（敌）/ 紫（朝贡）

点击具体一对关系 → 详细外交对话窗口。

### 4.4 外交对话窗口

```
┌─────────────────────────────┐
│  [玩家国旗] vs [对方国旗]    │
│  当前态度: -45               │
│  当前停战: 至公元前 350 年   │
├─────────────────────────────┤
│ 互动历史（编年体）          │
│  · -360 春，缔结互不侵犯     │
│  · -358 秋，秦伐魏河西       │
│  · -356 冬，议和，魏割河西    │
├─────────────────────────────┤
│ 可用动作：                   │
│ [缔约] [互市] [送质] [使节]  │
│ [挑衅] [宣战] [兼并主张]      │
└─────────────────────────────┘
```

### 4.5 战斗界面

战斗发生时**不**进入独立场景。地图保持，但弹出战斗叙事面板：

```

│  [秦] 白起 60万 vs [赵] 廉颇 45万 │
├───────────────────────────────────┤
│  地形：山谷 (赵 +15 防御)         │
│  阶段：对峙第 8 月                │
│  补给：秦 ▓▓▓▓▓ 赵 ▓░░           │
├───────────────────────────────────┤
│  叙事日志：                       │

└───────────────────────────────────┘
```

## 5. 视觉风格（Visual Style）

### 5.1 美学方向

**水墨工笔混合**：
- 地图远景：水墨写意，山川淡墨晕染
- 地图近景：工笔细描，城邑、关隘清晰
- UI 背景：仿古绢本纹理，半透明
- 边框：篆体纹路 / 云雷纹

### 5.2 配色

| 色彩定位 | 取色 | 用途 |
|---------|------|------|
| 主背景 | 米色（#F5EFD9） | UI 底色 |
| 次背景 | 古绢黄（#E8DBB0） | 面板内层 |
| 主文字 | 墨黑（#1A1A1A） | 正文 |
| 次文字 | 墨灰（#444444） | 辅助说明 |
| 强调（红） | 朱红（#B11E2B） | 警告、宣战 |
| 强调（金） | 鎏金（#C9A14A） | 胜利、王命、合法性 |
| 强调（青） | 黛青（#3A5A6A） | 信息、提示 |

> 所有 UI 元素避免使用纯黑、纯白、过饱和荧光色。

### 5.3 字体

| 用途 | 字体（建议） |
|------|------------|
| 标题 | 思源宋体 Heavy / 自定义篆楷 |
| 正文 | 思源宋体 Regular |
| 古典文本（事件、史书） | 思源宋体 Regular，加大行距 |
| 数字 / 西文 | Inter / 思源等宽 |

> 全部使用开源字体，避免商用授权问题。

### 5.4 动画

- **进入/退出**：淡入淡出 200ms，避免大幅滑动
- **数值变化**：滚动数字过渡 500ms，不弹跳
- **涂色变更**：渐变 300ms，加轻微音效
- **重大事件**：地图轻微闪烁 + 短暂震动（可在设置中关闭）

> "克制"是动画的最高美德。绝不使用浮夸的弹窗、闪光、屏幕震动。

### 5.5 音效与音乐

- **背景音乐**：古琴 / 编钟 / 笛 / 萧；分阶段（和平期、战时、危机、胜利）
- **音效**：克城（鼓）、收复（编钟）、外交（玉磬）、君主薨逝（编磬一声）
- **静默是默认**：玩家可关掉音乐保留音效，反之亦然

## 6. 操作（Controls）

### 6.1 鼠标

| 操作 | 行为 |
|------|------|
| 左键单击 | 选择 |
| 左键拖动 | 平移地图 / 框选 |
| 左键双击 | 聚焦缩放 |
| 右键单击 | 上下文菜单（驻军/调兵/外交...） |
| 右键拖动 | 旋转地图（如果支持） |
| 滚轮 | 缩放 |
| 中键 | 平移（可选） |

### 6.2 关键热键（默认）

| 键 | 行为 |
|----|------|
| **Space** | 暂停 / 恢复 |
| **+** / **-** | 加速 / 减速 |
| **1–8** | 切换主面板 |
| **F1** | 王宫 |
| **F2** | 军事 |
| **F3** | 外交 |
| **F4** | 内政 |
| **F5** | 快速存档 |
| **F9** | 快速读档 |
| **Esc** | 关闭当前面板 / 返回 |
| **Tab** | 切换地图模式 |
| **G** | 居中显示当前选中单位 |
| **B** | 显示战斗列表（如有进行中） |

> 所有热键应**可重映射**。

### 6.3 触屏（远期支持）

- v1 不主动支持触屏
- 但 UI 元素的命中区域应足够大（不要做 12px 的小图标），为未来铺垫

### 6.4 无障碍

- 色盲模式：势力色彩同时使用**纹理**区分（除了颜色）
- 字号：可在 80%–150% 之间调节
- 键盘可全控：所有功能可不用鼠标完成（远期目标）
- 高对比度模式：可选

## 7. 引导与教学（Onboarding）

### 7.1 新手剧本

为零基础玩家设计**专门的引导剧本**：
- 简化版地图（仅 3 国）
- 限定 30 分钟内的简短目标
- 关键操作的悬浮提示
- 不强制完成才能进入主剧本

### 7.2 渐进式提示

进入主剧本后，**首次出现**的概念会触发简短提示：
- "这是函谷关——它影响行军，详情见…"（点击查看详情）
- 提示一次后不再出现，但可在"百科"中查询

### 7.3 史书百科（Codex）

随时可打开的**内置百科**：
- 系统机制（"什么是合法性？"）
- 历史背景（"什么是三家分晋？"）
- 人物词条（点击任何历史人物可查）
- 操作教程

> 百科是非侵入的——玩家想要时点开，不想要时永远不出现。

## 8. 通知系统（Notifications）

### 8.1 通知样式

| 级别 | 视觉 | 持续 |
|------|------|------|
| L1 警报 | 居中弹窗 + 自动暂停 + 沉重音效 | 直到玩家关闭 |
| L2 提醒 | 顶部横幅 + 高亮闪烁 + 提示音 | 5 秒后入侧栏 |
| L3 通知 | 侧栏新增条目 + 静默音 | 永久保留 |
| L4 日志 | 仅写入事件日志 | 永久保留 |

### 8.2 通知聚合

同类事件 5 分钟内多次触发 → 聚合为单条（"边境 3 处发生小规模冲突"）。

### 8.3 玩家定制

设置中每类事件独立配置：
- 是否触发自动暂停
- 显示级别（L1–L4 升降）
- 是否伴随音效

## 9. 存档与读档界面

### 9.1 存档列表

- 显示**编年体摘要**（"秦昭襄王三年，东征魏河西"）
- 缩略地图（当时版图）
- 时间戳
- 玩家可命名

### 9.2 自动存档

- 每年年底自动存档（可关）
- 重大事件前自动存档（君主薨逝、宣战、变法、九鼎易主）
- 自动存档保留最近 N 个，旧的自动清理

详见 [`10-tech.md`](./10-tech.md) 存档实现。

## 10. 调试与开发模式（Dev Mode）

发布版默认隐藏，可由专门键启用：

- 显示 AI 当前国策
- 显示数据流
- 时间步进精确控制
- 强制触发事件
- 给指定势力赋予资源

> 这是开发与平衡测试的关键工具。

## 11. 待确认问题（Open Questions）

| # | 问题 | 备选 |
|---|------|------|
| OQ-1 | 是否首发支持中文 + 英文？ | 推荐：v1 先做中文，预留 i18n 架构；英文待社区或专业翻译 |
| OQ-2 | 战斗界面是否需要更"沙盘化"？ | 推荐：MVP 用文本叙事 + 关键参数即可；后期再考虑沙盘化呈现 |
| OQ-3 | 是否支持深色模式？ | 推荐：v1 先做"古绢"风格，深色模式作为后续可选 |
| OQ-4 | 是否做 PWA / 离线安装？ | 推荐：是。让玩家可"安装到桌面"，体验接近原生应用 |
| OQ-5 | 通知用哪种字体？ | 推荐：与正文同体系（思源宋体），仅大小有别 |
